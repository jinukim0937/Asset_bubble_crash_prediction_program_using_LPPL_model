# Asset bubble crash prediction program using LPPL model

## 개요

(주)쿼터백 자산운용의 요청에 따라 진행된 프로젝트로, 경제 물리 방법론인 LPPL을 사용하여 금융 상품의 버블 임계시점을 추정하는 GUI 프로그램을 개발하여 쿼터백자산운용사에 제공하는 프로젝트이다.

## 언어

MATLAB, C#

## 참여기관

연세대학교, (주)쿼터백자산운용, (주)Gilbreth

## 팀원

김진우, 김주엽

## 기간

2022년 05월 ~ 2023년 1월
